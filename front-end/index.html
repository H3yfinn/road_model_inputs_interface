    Metrics glossary & formulas

    • Branch Path: Hierarchy leading to the leaf node (e.g. Urban › Heating › Heat Pump).
    • Energy: Output energy in the chosen unit (PJ, GJ, or MJ).
    • Activity Level: Weights product along the path ($w_1^{\text{norm}} \times w_2^{\text{norm}} \times \dots$). Equal to $\text{Energy}_{\text{branch}} / \text{Energy}_{\text{total}}$.
    • Macro Driver: Metric and its active value. Falls back to first active driver (e.g., Households) if unassigned.
    • Energy Intensity: Calculated ratio of $\text{Energy} / \text{Driver Value}$.
    • Fuels: Visual list of fuels associated with the leaf node.

    Road Input Data and Provided Values

    Road input glossary

    • Core CSV fields: Branch Path, Variable, Region, Scale, Units, Per..., and 2022. The scenario is always Current Accounts.
    • Partial rows: Fine to leave some cells blank or omit non-essential columns entirely.
    • Ignored values: Blank or non-existent values are skipped by the loader.
    • Stock: Base-year vehicle count for a branch.
    • Mileage: Annual distance travelled per vehicle.
    • Final On-Road Fuel Economy: Energy used per distance travelled.
    • Passenger Vehicle Saturation: Passenger-road-only saturation shared across passenger road.
    • Vehicle Equivalent Weight: Weight used to convert vehicle types into equivalent vehicle activity.
    • Reconciliation Bound Lower: Minimum allowed road-sector reconciliation adjustment.
    • Reconciliation Bound Upper: Maximum allowed road-sector reconciliation adjustment.
    • Reconciliation Weight: Priority weight used when reconciling road-sector values.
    • Sales Share: New vehicle sales split by drive or fuel.
    • PHEV Electric Driving Share: Shared PHEV road-sector series.
    • Survival Rate: Road-sector age series for remaining vehicle vintages.
    • Vintage Profile Share: Road-sector age distribution used for the stock profile.
    • Blank fields: Export the grey provided base-year value unless you enter an override. Missing optional fields are ignored.

    Blank fields use the grey provided base-year value. Later-year changes are handled in LEAP adjustment variables, and missing optional values are ignored.
